# Blocks2D
This is my pet project to create a 2D block engine.

## Latest release
* **[Download](https://github.com/KykemanovDaniil/blocks2d/releases/latest)**

## Controls
* **W, A, S, D** - Player movement
* **I / O** - Zoom
* **F8** - Close game

## Tech Stack
* **Language:** C++ 23
* **Multimedia:** SFML 3.1
* **Generation:** FastNoiseLite
